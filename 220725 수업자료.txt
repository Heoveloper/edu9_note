자바스크립트
1. 변수 선언
	-const: 값이 초기화되면 재할당 불가
	const 변수명 = 값
	
	-let: 값이 초기화된 후 재할당 가능
	let 변수명 = 값
	
2. 문자열 탬플릿
	const weight = 100;
		case1)
		console.log('몸무게는 ' + weight + 'kg 입니다.');
		
		case2)
		console.log(`몸무게는 ${weight}kg 입니다.`);
	
	`${표현식}`
	표현식: 변수, 식, 함수(자바와 표현식 개념이 동일)
	
3. 함수
	-함수 선언문
	function 함수명(매개변수) {
		실행문;
	}
	-함수 표현식
	: 선언 후 호출 가능
	const 함수명 = function(매개변수) {
		실행문;
	}
	-화살표 함수(람다 표현식)
	: 주로 고차함수의 파라미터로 함수를 전달할 때 사용
	: 선언 후 호출 가능
	const 함수명 = (매개변수)=>{
		실행문;
	}
	
4. 전역변수/지역변수
	-전역변수: 최상위 함수 외부에 선언된 변수
	-지역변수: 함수 내부에 선언된 변수(매개변수 포함)
	
5. 전역객체 (window, globalThis)
	: 브라우저 실행환경에서 미리 생성된 내장객체

	콜백함수란? 고차함수의 매개값으로 전달하는 함수
