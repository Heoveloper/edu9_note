멤버 필드: 외부에서 정의해줄 수 없는 값

생성자 메소드: 외부에서 정의 가능
───────────────
return의 형태

return;	//현재 메소드 종료 후 호출한 곳으로 제어가 이동

return 표현식(값, 변수, 식, 함수 호출);	//현재 메소드 종료 후 반환값을 호출한 곳으로 전달
───────────────
instance 멤버(필드, 메소드) //class 개념에 정의된 instance는 멤버다.
static	 멤버
		1. 인스턴스 생성없이 클래스이름.멤버로 접근할 수 있다.
		2. 동일한 클래스로 생성된 인스턴스는 정적멤버를 공유한다.
//instance 속성값은 각자마다 다르고 static 속성은 공통이다.

class 개념{
	속성;
	
	생성자 메소드();

	행위; //행위 메소드를 통해서 instance 속성값이 바뀐다.
	
	getter();	//외부에서 정보 읽기 용도
	setter();	//외부에서 정보 수정 용도
}
